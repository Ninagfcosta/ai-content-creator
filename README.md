# NovaTech Apps — AI Content Creator

## What is this?
An AI system that generates unique, brand-aligned content 
for NovaTech Apps using OpenAI and two knowledge bases.

## How to run

1. Install dependencies:
pip install -r requirements.txt

2. Create .env file with your API key:
OPENAI_API_KEY=your_key_here

3. Run the app:
python src/main.py

## Knowledge Bases
- Primary: brand guidelines, product specs
- Secondary: market trends, competitor analysis

## Tech Stack
- Python 3.8+
- OpenAI API
- Markdown processing
- python-dotenv
